Glean provides powerful tools and integrations to help you build AI agents that can securely access and reason over your organization's knowledge. Our agent integrations enable you to create custom AI assistants that leverage Glean's enterprise search and chat capabilities.

## Getting Started

Choose an integration that best fits your needs:

### Model Context Protocol (MCP)
The MCP integration is ideal if you want to:
- Connect Glean directly to AI models like Claude or GPT-4
- Use Glean in MCP-compatible IDEs and tools
- Build agents that can securely access your organization's knowledge

[Learn more about Glean's MCP integration →](/agents/mcp)

### LangChain
The LangChain integration is perfect if you want to:
- Build agents using Python and the LangChain framework
- Create custom RAG applications with Glean as a retriever
- Integrate Glean into your existing LangChain applications

[Learn more about Glean's LangChain integration →](/agents/langchain)
