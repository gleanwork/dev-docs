## Welcome to Glean's Developer Platform

Glean is the Work AI platform that connects and understands enterprise data, to generate answers and automate work with AI. Glean simplifies building and scaling enterprise AI agents while offering you an open, extensible platform. Glean APIs enable you to connect enterprise data sources, customize and embed Glean into internal tools, and use Glean in the developer AI ecosystem.
### Why Build with Glean?


- **Search across all your enterprise data** - Glean builds hybrid search and a knowledge graph across enterprise data, with native support for 100+ connectors and APIs to index additional sources.
- **Build AI assistants and agents that get enterprise work done** - Create AI-powered applications grounded in your enterprise data, optimized for tasks like data analysis, expert discovery, and research.
- **Secure by design, built for the enterprise** - Glean protects your data with authenticated access, indexing controls, strict permissions enforcement, and support for single-tenant cloud environments.
- **Integrate with flexibility** - Use Glean's RESTful  APIs or client SDKs to build with the languages, frameworks, and developer tools you already useâ€” no need to change how you code.

Explore our documentation to learn how to implement Glean's capabilities in your applications and workflows.
